# MyXQL v0.8.2 - API Reference

- [MyXQL.GeometryCodec](MyXQL.GeometryCodec.md): Contract for encoding/decoding geometry types.
- [MyXQL.Queries](MyXQL.Queries.md): A struct for a prepared statement that returns multiple results.
- [MyXQL.Query](MyXQL.Query.md): A struct for a prepared statement that returns a single result.
- [MyXQL.Result](MyXQL.Result.md): Result struct returned from any successful query.
